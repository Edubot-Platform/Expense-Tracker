
##Expense Tracker

A responsive Expense Tracker web application built using HTML, CSS, JavaScript, and Bootstrap. This project helps users manage and track their income and expenses through a simple and user-friendly interface. The application is designed separately for both mobile and desktop devices to provide a better user experience across all screen sizes.

##Features
Responsive design for mobile and desktop
Add income and expense transactions
Display total income, total expense, and current balance
Dynamic transaction table
Delete transaction functionality
Attractive and modern UI design
Mobile-friendly layouts using media queries
Built using Bootstrap components
